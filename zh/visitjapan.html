在日本旅游的友善提醒

1. 准备日元现金

  • 1. 现金是必需的。
  • 2. 可用信用卡和二维码支付，但普及率约为60%。

2. 交通

  • 1. 在东京和大阪等主要城市使用公共交通工具。
  • 2. 主要交通工具：有轨电车、地铁、单轨列车、公共汽车和出租车。
  • 3. 可以租车。请注意，方向盘在右侧。
  • 4. 购买Suica卡对于乘坐公共交通很方便。
  • 5. 电车地铁的转乘信息：
    • (1) 日本换乘查询网站
    • (2) Yahoo Transit
    • (3) 东京交通地图 PDF
    • (4) 网上交通地图

3. 交通卡

购买交通卡后，您就不必每次使用公共交通时都购买票了。

  • 1. 从2023年6月起，Suica卡和PASMO卡将不再发售。持有日本信用卡的人可以通过手机App下载使用。
  • 2. 海外旅客应在机场购买Welcome Suica或PASMO PASSPORT 交通卡。
  • 3. Welcome Suica和PASMO PASSPORT卡的有效期为28天。您可以充值最高¥20,000日元，并可用于乘坐火车、地铁和公交车。
  • 4. 如果您离开日本时卡上还有余额，请在机场购物时使用。
  • 5. 更多信息：
  • (1) Welcome Suica： https://www.jreast.co.jp/multi/en/welcomesuica/
  • (2) PASMO PASSPORT： https://www.pasmo.co.jp/visitors/en/

4. 厕所

  • 1. 所有厕所均可免费使用。
  • 2. 在使用便利店厕所之前，请给店员打招呼。
  • 3. 厕所内提供卫生纸。
  • 4. 为保持卫生，请使用后务必冲水。

5. 吸烟和饮酒

  • 1. 20岁以下的个人禁止购买和使用香烟和酒精；这是法律规定，违反就是违法的,请注意。
  • 2. 在东京的街道上禁止吸烟。只允许在指定区域吸烟。
  • 3. 吸烟后，请务必将烟蒂放入便携式烟灰袋。便携式烟灰袋可在便利店购买。
  • 4. 理性饮酒，切勿酒后驾车。

6. 垃圾分类

  • 1. 确保正确分类和处理垃圾。
  • 2. 资源垃圾：饮料瓶，罐头，玻璃瓶。可以把饮料瓶放在便利店的回收箱。
  • 3. 可燃垃圾：一般家庭垃圾，纸张和衣物。
  • 4. 不可燃垃圾：金属，玻璃，陶瓷和鞋子。
  • 5. 大件垃圾：边长30厘米或更长的物品，例如手提箱。注意：大型垃圾需要收费，如行李箱坏了需要付费处理的，请联系房东。
  • 6. 千万不要将 资源垃圾如饮料瓶放入到可燃垃圾中。这将导致可燃垃圾不被收走，也会给房东造成麻烦。

7. 日本的电源插座

  • 1. 日本家庭的电压为110V。
  • 2. 日本电源插座的类型是A型，最大电流为15A。
  • 3. 请事先购买并携带海外转换插头。
  • 4. 您也可以在便利店购买智能手机充电器。
